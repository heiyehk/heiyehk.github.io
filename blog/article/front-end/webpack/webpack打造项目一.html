# webpack打造项目(一)-基础创建

# 前言

基础创建，配置webpack文件。使用webpack-dev-serve启动项目，更改文件实时更新。配置scripts命令化启动。
webpack中文文档：https://www.webpackjs.com/guides/getting-started/

# -i 的意思是install
npm -i moduleName # 安装模块到项目目录下
npm -i -g moduleName # -g 的意思是将模块安装到全局，具体安装到磁盘哪个位置，要看 npm config prefix 的位置。
npm -i -S moduleName # -S 的意思是 -save 将模块安装到项目目录下，并在package文件的dependencies节点写入依赖，简单点就是开发环境要用的。
npm -i -D moduleName # -D 的意思是 --save-dev 的意思是将模块安装到项目目录下，并在package文件的devDependencies节点写入依赖，正式环境要用的。

# 开始创建

安装全局webpack

npm i webpack -g
1

然后在所需要创建的目录中创建文件夹mkdir createapi && cd createapi，进行npm初始化，这里可以使用npm init -y直接一步到位，-y就是所有参数yes的意思，不加则自己配置

npm init # 在项目下会出现一个packge.json，这里不过多赘述
1

然后安装局部webpack，webpack 4+ 版本需要安装 CLI。

npm i webpack webpack-cli -D
1

当前根目录结构

|-package-lock.json
|-package.json

webpack在执行时，除在命令行传入参数，还可以通过指定的配置文件来执行。默认会搜索当前目录下webpack.config.js。这个文件是一个node.js模块，返回一个json格式的配置对象，或者通过--config选项来指定配置文件。在根目录下创建webpack.config.js（webpack配置文件）、index.html（页面）、main.js（入口文件）。在使用html时需要引入插件html-webpack-plugin

npm i html-webpack-plugin -S
1

# webpack.config.js中配置入口出口以及页面配置信息

const HtmlWebpackPlugin = require('html-webpack-plugin'); // 使用nodewebpack可以在类似Node.js的环境中进行编译（使用Node.js require加载块，而不接触任何内置模块，例如fs或path）。
const path = require('path')

const config = {
  target: 'web',
  entry: './main.js', // 输入文件
  output: {
    filename: 'bundle.js', // 输出文件
    path: path.join(__dirname, 'dist') // 输出路径，这里需要使用绝对路径，否则不能打包
  },
  mode: 'development', // 设置环境
  plugins: [
    new HtmlWebpackPlugin({
      template: path.join(__dirname, 'index.html'), // 注入html文件
      template: 'index.html',
      inject: true // 注入选项
    })
  ]
};

# webpack-dev-serve

这webpack-dev-server为您提供了一个简单的Web服务器以及使用实时重新加载的功能。 webpack-dev-server中文文档：https://www.webpackjs.com/guides/development/#使用-webpack-dev-server

npm i -D webpack-dev-serve
1

# webpack.config.js

修改配置信息

const HtmlWebpackPlugin = require('html-webpack-plugin'); // 使用nodewebpack可以在类似Node.js的环境中进行编译（使用Node.js require加载块，而不接触任何内置模块，例如fs或path）。
const path = require('path');
const config = {
  target: 'web',
  entry: './main.js', // 输入文件
  output: {
    filename: 'bundle.js', // 输出文件
    path: path.join(__dirname, 'dist') // 输出路径，这里需要使用绝对路径，否则不能打包
  },
  mode: 'development', // 设置环境
  devServer: {
    contentBase: './dist'
  },
  plugins: [
    new HtmlWebpackPlugin({
      title: '测试',
      filename: 'index.html',
      minify: true, // 是否对html文件压缩
      template: path.join(__dirname, '../public/index.html'), // 注入html文件
      inject: true, // 注入选项
      hash: true // hash 值是该次 webpack 编译的 hash 值
    })
  ]
};
